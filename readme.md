#Recipe service

##global req
1. Python 3.8+
2. PostgeSQL 10+

##install
1. clone this repo
2. Install dependencies
    ```
    pip install -r requirements.txt
    ```
3. *[optional]* run 
    ```shell script
    python initial_data.py
    ```
   for create DB schema and load demo fixtures
4. Start server:
    ```shell script
    python run.py
    ```

##API Docs
Swagger documentation are available there: http://localhost:8000/api/docs#/
